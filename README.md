🚀 Portfólio Pessoal - Pedro Silva
Este projeto é o meu portfólio profissional, desenvolvido para consolidar meus conhecimentos em Front-end (HTML5, CSS3, JavaScript) e UI/UX Design. O design foi totalmente prototipado no Figma, focando em uma estética moderna e limpa.

🛠️ Tecnologias Utilizadas
HTML5: Estruturação semântica.

CSS3: Estilização avançada com Flexbox e variáveis.

Figma: Prototipagem e Design de Interface.

Google Fonts: Tipografia "Anybody" para uma identidade visual forte.

🎨 O Projeto
O objetivo deste portfólio é apresentar meus projetos de desenvolvimento web e serviços de UI/UX, unindo a clareza técnica com a funcionalidade do design.
